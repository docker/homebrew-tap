# Docker Tap

## How do I install these formulae?

`brew install docker/tap/<formula>`

Or `brew tap docker/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "docker/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
